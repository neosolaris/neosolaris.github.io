## Intro

* [FITS](fits) (Flexible Image Transport System) 파일 포맷을 보거나 편집할 수 있는 프로그램
* fits 포맷은 주로 천문사진에서 사용되는 포맷이다.
* 리눅스의 경우 직접 다운로드 받는 것보다 패키지로 설치하는 것이 좋다.

## Install

* ubuntu install
```console
$ sudo apt install wish ftools-fv 
$ fv
```
