## INSTALL

```console
git clone https://github.com/mmistakes/minimal-mistakes.git
cd minimal-mistakes/
echo 'gem "webrick"' >> Gemfile
bundle install
bundle exec jeckyll serve --host 127.0.0.1 --port 4000
```

vim _posts/my-first-post.md
