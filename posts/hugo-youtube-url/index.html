소개#

  • hugo에서 지원되는 youtube 영상 포함 문법은 다음과 같다.
  • 문제는 브라우저에서 동영상 URL을 복사하면 Youtube-ID만 추출하여 넣어 주는 것이 무척 번거롭다.
  • 이를 해결하기 위해서는 다음의 세가지 방법이 있다.
    1. 브라우저 주소창에서 직접 ID부분만 영역 복사하기
    2. 브라우저의 북마클릿(Bookmarklet)을 javascript로 만들어 ID 복사하기
    3. 복사한 URL을 입력하면 자동 추출해주는 프로그램 만들기

Brave(또는 Chrome)에서 북마클릿을 이용한 유튜브 URL ID복사#

  1. Brave 브라우저에서 북마크 추가
  2. Name: Youtube-ID-Copy
  3. URL: 다음 코드 입력 후 저장
javascript:(function(){
  const url = window.location.href;
  const match = url.match(/(?:v=|youtu\.be\/)([A-Za-z0-9_\-]{11})/);
  if(match){
    const id = match[1];
    navigator.clipboard.writeText(id);
    alert("영상 ID 복사됨: " + id);
  } else {
    alert("영상 ID를 찾을 수 없습니다.");
  }
})();
