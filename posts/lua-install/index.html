Install#

  • 설치 - luaJit과의 호환을 위해 5.1로 설치.
sudo apt install lua5.1
  • lua -v - version
  • lua -h - help
  • lua -i - interactive mode

Hello World#

  • hello-world.lua
#!/usr/bin/env lua
print("Hello World!")
  • 실행
chmod u+x hello-world.lua
./hello-world.lua

$ lua
