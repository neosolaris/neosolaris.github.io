Usage#

$ ncmpcpp
  • F1 : show help
  • 1 : show playlist ;
  • 2 : show directory browser
  • 3 : show search
  • 4 : show library
  • 5 : playlist editor
  • 6 : tags editor
  • 8 : visualizer
  • p : toggle play/pause
  • a : add selection to playlist
  • > : play next track
  • < :play previous track
