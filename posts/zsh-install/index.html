zsh install and setup

2022-09-02 · 59 words

zsh 설치#

$ sudo apt install zsh

기본쉘 변경#

$ chsh -s /usr/bin/zsh

Oh-My-Zsh 설치#

$ sh -c "$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

테마변경#

$ vim ~/.zshrc
$ source ~/.zsh
  • .zshrc 설정 변경
ZSH_THEME="agnoster"

해결되지 않은 문제#

  • 기본쉘을 zsh로 했을 때 한글 입력이 되지 않는다.
  • bash의 서브쉘로 실행했을 때는 한글 입력이 잘 된다.
