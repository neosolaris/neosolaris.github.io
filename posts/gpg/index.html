Encrypt and Decrypt Messages with GPG#

$ ls

Key Maintenance#

$ gpg --list-keys    # list available GPG keys
$ gpg --refresh-keys # update the key information
$ gpg --keyserver <key_server> --refresh-keys  # pull information from a specific key server
